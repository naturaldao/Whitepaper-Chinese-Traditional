# 概述

道易程是以太坊通證經濟和DAO的價值體系的公共治理層。

這意味着道易程是一個讓 DAO暨dApp 繁衍生息，讓通證經濟蓬勃發展的公共治理層。

道易程即將引領我們邁入舉世矚目的元宇宙元器件時代！

道易程的開發始於2019年4月1日。
